🌍 AI Travel Agent Chatbot

An intelligent Travel Planning Assistant built using Google ADK (Agent Development Kit) and Groq Llama 3.3 70B.
The chatbot helps users plan personalized trips by providing destination recommendations, itineraries, budgeting guidance, accommodation suggestions, transportation options, and travel tips.

✨ Features
Personalized travel recommendations
Budget-based destination suggestions
Detailed day-by-day itinerary generation
Travel cost estimation
Accommodation recommendations
Food and local experience suggestions
Transportation guidance
Packing recommendations
Safety and travel tips
Destination comparison support
Interactive follow-up questions for better trip planning


🛠 Tech Stack
Python
Google ADK
Groq API
Llama 3.3 70B Versatile
LiteLLM

📋 Response Format

The AI Travel Agent provides responses in the following structure:

🌍 Trip Summary
📅 Suggested Itinerary
💰 Budget Breakdown
🏨 Accommodation Options
🍴 Food Recommendations
🚗 Transportation
🎒 Packing Tips
⚠️ Important Travel Notes
